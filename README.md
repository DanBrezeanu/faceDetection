# faceDetection
Face recognition pyscripts

```
Syntax

python <script>.py -p PROTOTXT -m MODEL -c CONFIDENCE [-v VIDEO | -i IMAGE]

Options:
  -p, --prototxt
      Path to prototxt file
      
  -m, --model
      Path to pretrained caffemodel file
      
  -c, --confidence
      Lower threshold for confidence level
      Default: 0.5
      
  -v, --video
      Path to video file
      
  -i, --image
      Path to image file
```
